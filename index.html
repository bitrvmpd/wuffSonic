wuffSonic

Another Subsonic API written in C#. It's a WIP.

Nuget Package now available.

How to use it:

    try 
    {
        Credentials c = new Credentials(
                appName: "yourAppName",
                user: "yourUser",
                password: "yourPassword",
                version: "1.11",
                uri: "http://music.haus.foxburu.mx"
                );
        Ping p = new Ping()
        {
            Credentials = c
        };
        //Call Ping
        var response = await p.DoRequest<PingResponse>();
    }
    catch(SubsonicException se)
    {
        Console.WriteLine(se.Error.Details.message);
    }
